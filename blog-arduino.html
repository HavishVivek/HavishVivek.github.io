Setting up Arduino-cli

Introduction

In this tutorial, let's explore the setup process of Arduino CLI in the Mac OS. The process just takes only 10 minutes

Installing Homebrew

Open up a terminal window and install homebrew with the following command:

Terminal 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Adding Homebrew to the path

Terminal 
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> /Users/[username]/.zprofile
Terminal

Terminal
